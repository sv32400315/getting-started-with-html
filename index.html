World Wide Web
-collection of HTML (HyperText Markup Language) documents
-glues everything together

Major Pieces of the Web
HTTP- the main protocol of the web
Servers-computers that host the files that make up the web
Internet- the world's largest computer
Browser-a program that runs on your computer to display files found on the web. Examples are Internet Explorer, Chrome and Firefox

HyperText Markup Language (HTML)
-HTML documents contain text content which describes "what you see" and markup which describes "how it looks"
-references to other documents. e.g. images and videos
-links to other pages
-during the lesson, the instructor used a simple HTML editor called HTML Playground
-Plain text is plain text in HTML
-HTML markups made up of tags.
-An HTML tag is always contained within angled brackets with both an Opening and closing tag. There is an opening and closing tag. The closing tag has a slash.
An HTML element refers to everything within a set of opening and closing tags
-An HTML attribute is a property of an HTML element. The href attribute is set to an anchor tag for the URL

HTML Tags
-no closing tag is needed if no content
-a tag is an anchor tag. Anchors are for making links
-b tag is used to bold content
-em tag is used to italicize. em stands for emphasize
-img tag are for images. alt text is used in case the image does not load. Alt stands for alternate and is considered a void tag.
-Images do not need closing tags
-br tag stands for break and used to create multiple lines
-p tag stands for paragraph. The p tag creates an invisible box with height and width
-anchor tags use the href attribute
-img tags use the src attribute
-Inline and Block. Inline are a,br, b, em, img, strong and span. Block are p, form and div.
-Span and div both can have content. They do not do anything but contain their content. Div act more like a paragraph tag

HTML Document
-Complete HTML document has a doctype, opening HTML tag, closing HTML tag, head and body
-Doctype tell what kind of HTML it is. Used to be more complicated. Now with HTML 5, it is nice and simple
-The section called Head includes meta data and other random stuff. May also include Java script and CSS. In addition, may include title of thpage which is a the top of the browser window
-we have been working on the body tag in the lesson. Focusing on the content that will be between the body tags

Concepts to Remember
- How the Web Works: The web is a bunch of computers that communicate with each other. When a person goes to a web page like www.google.com, their computer sends a HTTP Request to a server. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen
- HTML: HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks"
-Tags and Elements: HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags
-Why Computers are Stupid: Computers are stupid because they interpret instructions literally. This makes them very unforgiving since a small mistake by a programmer can cause huge problems in a program.
-Inline vs Block Elements: HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them
